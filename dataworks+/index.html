For further study I learned that providing a border radius in px and % have different affects on the curve. If you provide a border radius of 50% it will create an oval, curving every part of the border. If you provide a border radius of 50px it will create a semi-circles that is 50px in diameter on each end of the border. I'm not sure if I'm correct about this but the button is 8rem x 3rem, so a radius of 1.5rem would be 50% of the height but when I specify 50% the curve is different. But if I set the radius to 4rem (50% height) the curve is correct again. So it seems an absolute value will be constrained by 1/2 the width/height of the box but a relative one will affect the entire curve, crating an oval. I see nothing in the docs indicating this to be the case.
